# MusicBrainz GraphQL API

MusicBrainz is an open music encyclopedia. The API covers artist lookups, recording metadata, releases and release groups, labels, works, places, events, instruments, series, and relation data between music entities.

**Endpoint:** No native GraphQL endpoint — conceptual schema derived from MusicBrainz REST API
**Documentation:** https://musicbrainz.org/doc/MusicBrainz_API
